# Package `CentOS 7, 8`


## Initialization
[ShellScript `centos.sh`](./centos.sh.md)
```shell
./centos.sh add_user <user> <password>
./centos.sh add_group <user> <group>
./centos.sh add_sshkey <user> <pubilc_key>
```
